How to Update

IMPORTANT : Make a backup for your website (files and database) before making this update.

  • Login to your FTP account
  • Replace all files, except the files below:
.env
storage/installed
public/assets/css/style/custom.css
public/uploads (folder)
  • Open routes/web.php file
  • Place this code in it (Before all other command lines)
Route::get('update', function () {
    \File::delete(app_path('Larapen/Helpers/functions.php'));
    \File::delete(app_path('Larapen/Helpers/wordpress.php'));
    \File::delete(app_path('Larapen/Models/Pack.php'));
    \File::moveDirectory(public_path('vendor/adminlte/plugins/jquery/'), public_path('vendor/adminlte/plugins/jQuery/'));
    
    $updateSqlFile = storage_path('database/updates/from-2.8-to-2.9/update.sql');
    if (file_exists($updateSqlFile)) {
        $sql = file_get_contents($updateSqlFile);
        $sql = str_replace('<<prefix>>', \DB::getTablePrefix(), $sql);
        \DB::unprepared($sql);
    }
    
    $exitCode = Artisan::call('view:clear');
    
    return 'Success';
});
  • Visit yoursite.com/update to migrate your website
  • Remove the code after update.
  • You are done.
